# motux-fomod

motux-fomod is a library for parsing and evaluating [FOMOD](https://fomod-docs.readthedocs.io/) mod installer configurations.

FOMOD is an XML-based format used by mod managers (Mod Organizer 2, Vortex, etc.) to define guided installation wizards for game mods.
